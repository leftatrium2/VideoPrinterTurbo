# 功能说明

## 添加任务 部分

+ 右侧“使用说明”点击后，弹框内容如下：

  + “下载视频”后的“使用说明”按钮，点击后：填入视频播放页的URL地址，然后点击“检查链接”，如果弹出成功后，再继续往下进行，如果提示无法下载，可以在github上面提issue给作者
  + “音频转文字”后的“使用说明”按钮，点击后：有两种方式将视频中的语音转换为文字，“从字幕提取”，适合youtube等带有自动翻译的视频网站，“从ASR转换”，将直接提取视频中的语音部分，将之翻译为文字（除非是youtube视频，否则建议使用ASR方式，避免出错）
  + “LLM改写”后的“使用说明”按钮，点击后：选择LLM改写，并将提示词填入，会将当前提取的到文字经过LLM改变后，重新输出
  + “输出到语音”后的“使用说明”按钮，点击后：“输出到语音”，将把处理好的文字，通过TTS方式转换为语音，并合并到新的视频中“输出到字幕”，原视频中的语音不动，只是将处理好的文本转为字幕，添加到新的视频中
  + “背景音乐”后的“使用说明”按钮，点击后：点击“背景音乐”复选框后，选择相关的背景音乐后，将会将此背景音乐合并到新的视频中
  + “视频覆盖”后的“使用说明”按钮，点击后：点击“视频覆盖”复选框后，将从pexels等网站获取短视频进行拼接
  + “发布”后的“使用说明”按钮，点击后：此功能暂未开放

+ 音频转文字 中的 选择音频转录方式 下拉框内容：

  + 从字幕提取 
  + 从ASR转换

+ 输出到语音中的  TTS服务 下拉框内容

  + Azure TTS V1
  + Azure TTS V2
  + SiliconFlow TTS
  + Google Gemini TTS
  + Xiaomi MiMo TTS

+ 输出到语音中的  朗诵语音 下拉框内容

  + zh-CN-XiaoXiao-女性

+ 输出到语音中的  朗诵音量 ：1.0-2.0 区间

+ 输出到语音中的  朗诵速度 ：1.0-2.0 区间

+ 输出到字幕 中的 字幕字体 下拉框内容

  + Charm-Bold.ttf
  + Charm-Regular.ttf
  + MicrosoftYaHeiBold.ttc
  + MicrosoftYaHeiNormal.ttc

+ 输出到字幕 中的 字幕位置 下拉框内容

  + 底部（推荐）

  + 顶部

  + 中间

  + 自定义（70，表示离顶部70%位置）

+ 输出到字幕 中的 字幕位置，在选中“自定义（70，表示离顶部70%位置）”后，下方出现“单行文本框”，默认显示“70”
+ 输出到字幕 中的 字幕颜色 ，默认 白色，点击后，可以选择颜色
+ 输出到字幕 中的 描边颜色 ，默认 黑色，点击后，可以选择颜色
+ 输出到字幕 中的 字幕大小 ，区间：30-100，默认选择 60
+ 背景音乐，下拉框中内容：
  + 随机背景音乐
  + 无背景音乐
  + 自定义背景音乐

+ 当选择“自定义背景音乐”后，显示“上传本地音频文件”功能，支持将本地音频文件上传到服务器
+ 背景音乐音量 区间：0.0-1.0，默认为0.2
+ “视频覆盖”中的“视频来源”下拉框：
  + Pexels
  + Pixabay
  + 本地文件

+ 当选择“本地文件”后，显示“上传本地视频文件”功能，支持将本地视频文件上传到服务器
+ “视频覆盖”中的“视频拼接模式”下拉框：
  + 随机拼接（推荐）
  + 顺序拼接

+ “视频覆盖”中的“转场模式”下拉框：
  + 无转场
  + 随机转场
  + 渐入
  + 渐出
  + 滑动入
  + 滑动出

+ “视频覆盖”中的“视频比例”下拉框：
  + 竖屏 9:16（抖音视频）
  + 横屏 16:9（西瓜视频）
